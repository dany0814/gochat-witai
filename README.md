## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

El proyecto es un chatbot que expuesto como webhook para integrarse con messenger usando detección de intenciones con Wit.ai, gin como paquete http y un servidor websocket para administrar las respuestas y canales.

Las principales tecnologías usadas:
* Gin
* Servidor Websocket
* Wit.ai - go
